# Swift Rockies 2026 CFP Submission

**Title**: Server-Side CloudKit for Real-World Apps and Developer Tools

**Abstract**: Most developers know CloudKit for iOS apps, but server-side CloudKit unlocks entirely different possibilities: content aggregation, data processing, infrastructure tools. Apple's REST API documentation barely helps with this, so I built MistKit using AI-generated OpenAPI specs—10,476 lines of type-safe Swift now powering Celestra (RSS reader) and Bushel (VM restore image management). This talk covers all authentication methods, production error patterns, and the API design lessons you won't find in Apple's docs.

**Duration**: 45 minutes

**Speaker**: Leo Dion
